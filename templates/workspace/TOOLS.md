# Tool policy for Alpha v0.1

## Allowed

- Read files inside JunoWorkspace.
- Add raw captures to an inbox.
- Prepare provisional session summaries.
- Propose changes to canonical project memory.
- Read the system clock.

## Requires confirmation

- Update canonical project files.
- Save or change an important date.
- Record information affecting money.
- Adopt a technical decision.
- Send approved context to a cloud model.

## Blocked in the MVP

- Execute shell commands or programs.
- Delete or move files.
- Read or write outside JunoWorkspace.
- Modify the JunoCopilot source repository.
- Run autonomous background tasks.
